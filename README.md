# Instructions
1. Install Python **3.7**
2. Install pygame
    ```bash
        pip install pygame (or)
        pip3 install pygame
    ```
3. Install neat
    ```bash
        pip install neat
    ```
4. Download the game folder
5. And simply run the game.py file to see the performance of AI
